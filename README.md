# NovaUI Experience

A modern multi-page responsive web application built using HTML, CSS, and JavaScript with interactive UI components, dynamic task management, and responsive layouts.

NovaUI Experience focuses on creating a clean and visually engaging frontend experience while implementing core web development concepts such as form validation, DOM manipulation, CRUD operations, Flexbox, CSS Grid, and responsive design.

---

# ✨ Features

## 🏠 Home Page
- Modern hero section
- Smooth navigation
- Interactive buttons
- Theme switching
- Responsive layout
- Animated UI cards

---

## 📞 Contact Form
A fully styled contact form with:
- First Name
- Middle Name
- Last Name
- Email Address
- Phone Number
- Message Box

### ✅ Validation Features
- Required field validation
- Valid email format checking
- Phone number validation
- Real-time error messages
- JavaScript-based form handling

---

## ✅ Dynamic To-Do List
Interactive task manager with CRUD functionality.

### Features
- Add Tasks
- Edit Tasks
- Delete Tasks
- Dynamic DOM Manipulation
- Task timestamps
- Responsive task cards
